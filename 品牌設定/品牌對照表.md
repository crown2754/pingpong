# 品牌對照表

小說內品牌均為現實桌球品牌的虛構對應名，正文統一使用左欄名稱。

| 小說品牌 | 現實範本 | 簡述 |
|----------|----------|------|
| **蜜蜂** | Butterfly（蝴蝶） | 日系大牌；膠皮、底板、球衣；森夫後期常逛 |
| **米他庫** | Nittaku（尼塔庫） | 日系；底板紮實、賽事用球常見 |
| **多米克** | Donic | 德系；性價比、經典系列多 |
| **雙子** | 雙魚（Double Fish） | 中國品牌；入門到進階皆有 |
| **VISTA** | VICTAS | 日系；膠皮線完整，V>15 等為高階象徵 |
| **STIKA** | Stiga | 瑞典系經典；Clipper、Carbon 系列知名 |
| **Teebar** | Tibhar | 德系；Evolution、Rapid 等膠皮常見 |
| **Boola** | Joola | 德系；近年贊助與配色醒目 |
| **藍精靈** | 紅雙喜（DHS） | 中國國球品牌；狂飆系列為正手經典 |
| **Xim** | Xiom | 韓系；歐若拉、狂飆替代款討論度高 |
| **宇宙** | 銀河（Yinhe） | 膠皮線；森夫第一套用 **月球速度** |
| **矮豆** | IDOL（卓隆／郗恩庭） | 台灣代理常見；森夫第一支底板 **極光3** |

## 森夫器材路線（備註）

| 階段 | 取向 |
|------|------|
| 第一套（第二章） | **矮豆 極光3（FL）** + **宇宙 月球速度** 中軟（正手黑／反手紅）；為改善僵硬、體驗咬球 |
| 第一季末（第九章） | 試打莊志淵特注拍；確立 **狂暴三**／**X05** + **快速反擊** |
| 第二季 | **多米克 紅張（FL）** + 狂暴三／X05；**Teebar** 免綁帶桌球鞋 |
| 後期 | 米他庫、VISTA、Xim 等持續研究 |
