# 宇宙

| 項目 | 內容 |
|------|------|
| 現實範本 | 銀河（Yinhe） |
| 定位 | 入門至中階、性價比、練球不心疼 |
| 森夫關係 | **第一套膠皮**品牌 |

## 小說常用型號（虛構對應）

| 型號 | 說明 |
|------|------|
| **宇宙 月球速度**（中軟） | 範本 **銀河 月球速度**；正手**黑**、反手**紅**，兩面皆中軟 |

## 森夫選用理由（第二章）

- 身體僵硬、重啟後需要**咬球、擊出回饋**的感覺；中軟比硬膠皮好發力、好體驗包球。
- 兩面同款、僅顏色區分，沾膠皮時較不易搞混正反手特性（但仍可能貼反顏色）。
- 價格可控，留預算給 **矮豆** 底板。

## 備註

- 舊設定 **宇宙 U-2** 底板已廢；森夫第一支底板改為 **矮豆 極光3**。
