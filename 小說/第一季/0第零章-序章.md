# 序章：褪色的球拍

**2026 年 5 月 15 日（五）｜ 21:17**

陳森夫的耳機裡還有人說話。

螢幕上是三個時區的臉孔，其中一個在分享簡報，簡報上的字他看得懂，意思卻像隔了一層水。會議室其實是他的書房——嚴格說，是客廳角落那張 IKEA 書桌。娜娜剛洗完澡，穿著睡衣從他身後經過，手裡端著一杯溫熱的豆漿，輕輕放在桌角。她沒出聲，只是用手指點了點他的肩，像提醒，也像心疼。

森夫點了一下頭，嘴型說：快好了。

他們結婚不到半年。交往很多年，登記卻拖到今年春天——說起來有點好笑，兩個三十幾歲的人，為了避開辦公室裡的閒話，當初還特地一起離職，各自換了公司。從上司與部屬，變成丈夫與妻子。娜娜說，這樣比較像重新開始。森夫覺得她說得對，雖然他不太會把「覺得對」說出口。

「森，這個 API 行為跟 spec 不一致，你能不能再確認一下？」

「……好，我明天早上前回。」

他關掉麥克風，簡報定格在一張他早已看膩的架構圖上。三十五歲，全端工程師，身高一百七十七，體重九十五公斤——上個月體檢，護理師唸數字的時候語氣很客氣，客氣本身就是一種警告。娜娜說沒關係，慢慢減。森夫說嗯。然後繼續加班。

會議終於結束時，窗外下了雨。

台北的雨來得沒什麼預告，街燈在水氣裡糊成一片。森夫合上筆電，娜娜已經在沙發上睡着了，手機還亮著，播到一半的戀愛綜藝。他替她關掉螢幕，蓋好薄毯，關了客廳主燈。結婚後他常有一種感覺：家裡很暖，自己卻還是累的，累到骨頭裡那種。

他原本也該睡了。

但肚子餓，冰箱裡只剩半盒豆腐和過期的優格。他抓了鑰匙和傘出門，想去巷口那間還開著的便利商店買個飯糰。樓下電梯很慢，鏡子照出他——POLO 衫、拖鞋、肩頭微塌——一張台北市到處可見的臉。

雨小了一點。

他走在忠孝東路某段不會被觀光客記住的人行道上。紅燈。路邊有一棟老舊大樓，一樓是乾洗店，旁邊有扇往下走的鐵門，漆成褪色的綠。門上掛著手寫木板：**莊武鴛 桌球館**。

森夫本來不會注意。

但風把雨絲斜斜吹過來，同時從那扇鐵門底下——或者說，從地底下——傳來兩聲乾脆的撞擊。

乒。

乓。

他的腳步停了。

不是因為好奇，比較像身體比腦子先反應。肩膀微微沉下去，右腳往前半步，膝蓋彎了一點——那是很多年前，學長喊「森夫，接球」的時候，肌肉自己記得的姿勢。他站在原地，雨傘上的水珠一顆顆滑落，滴答，滴答，和地下室裡又一輪擊球聲疊在一起。

大學一年級，他報了桌球社。

教練說他骨架還可以，讓他跟著校隊練了一年。實際上多半是撿球、擺球檯、看學長們對攻。他學會正手握拍，知道旋轉不是玄學，也曾在某個下午，連續十板正手對攻沒有漏——學長說，欸，可以喔。就那麼一句，他高興了整整三天。

後來專題、實習、碩士、求職。球拍塞進書桌最下層抽屜，膠皮慢慢變黏、變硬、變成另一種時間。他沒有覺得可惜。那時候覺得，人生有更重要的事。

——現在呢？

現在他在雨裡站了不知道多久，直到乾洗店的老闆娘探頭出來收衣架，狐疑地看他一眼。

森夫回過神，低聲說了句沒關係，撐開傘繼續往便利商店走。買了最喜歡的芋頭口味飯糰，又多拿了一瓶無糖茶。結帳的時候，他卻一直想起那兩個音節。乒、乓。像有人用指節敲在他的胸口，不重，卻一下一下，敲得他空蕩蕩的肚子裡有什麼東西醒過來。

回家時娜娜還在睡。他坐在床邊吃飯糰，油膩的塑膠膜聲在安靜的臥室裡顯得很大。吃完，他沒有立刻躺下，而是打開了陽台旁那口落滿灰的儲物櫃。

最下層。

一只舊球拍袋，拉鍊已經澀了。他拉開，聞到一股膠皮與塵埃混合的味道——很舊，卻莫名熟悉。拍柄上還貼著他大學時寫的姓名貼，邊角翹起：**陳森夫**。

膠皮裂了。海綿大概也死了。

他握起球拍，在黑暗的臥室裡空揮了一下。風切過拍面，什麼也沒有，只有空氣。娜娜在夢裡翻了個身，含糊地喊了一聲老公。

森夫把球拍放回膝上，低聲說：

「……明天再說。」

窗外雨又大了。

他不知道的是，隔牆鄰居許家偉的客廳燈也還亮著，螢幕上是紅綠跳動的 K 線。他們兩個人的夜，一個在舊球拍前坐下，一個在期貨圖前發呆——都還沒有交集。

但巷口那間 **莊武鴛 桌球館** 的燈，會一直亮到很晚。

而有些人的人生，就是從聽見一聲「乒」開始，再次滾動的。
