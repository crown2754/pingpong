# 序章：褪色的球拍

**2026 年 5 月 15 日（五）｜ 21:17**

陳森夫的藍牙耳機裡，跨國會議還在繼續。

筆電螢幕上擠著三個不同時區的疲憊臉孔。其中一人正在分享簡報，那些密密麻麻的英文字母與數據架構，森夫每個都看得懂，但此時刻在眼底，卻像隔著一層泛著白光的溫水，怎麼也進不到腦袋裡。

這間「線上跨國會議室」，其實只是他家客廳角落的一張 IKEA 書桌。剛洗完澡的娜娜穿著舒適的棉質睡衣，輕手輕腳地走到他身後。一聲沒吭，只是把一杯溫熱的豆漿輕輕放在桌角，順手在他的肩膀上捏了捏——那力道帶著提醒，更多的卻是心疼。

森夫側過頭點了點，用嘴型無聲地說：快好了。

他們結婚還不到半年。交往多年，登記卻拖到了今年春天。說起來有點諷刺，兩個三十多歲的上班族，為了避開辦公室裡那些無聊的閒言閒語，當初還特地一起離職，各自換了新公司。從職場上的「上司與部屬」，變成了同一個屋簷下的「丈夫與妻子」。

娜娜那時說，換個環境，兩個人才像重新開始。森夫心裡覺得她說得太對了，但他向來不是個擅長把情感說出口的人。

「森，這個 API 的行為跟 spec 不一致，明天上線前能再確認一下嗎？」耳機裡傳來專案經理毫無起伏的聲音。

「……好，我明天早上前回。」

他熟練地切斷麥克風。簡報定格在一張他早已看膩的系統架構圖上。反射在螢幕白光裡的，是一個三十五歲、全端工程師、身高一百七十七公分、體重九十五公斤的肥胖身影。上個月體檢時，護理師覆誦體重數字的客氣語氣，至今想起來都像是一種溫柔的警告。

娜娜當時笑著說沒關係，我們慢慢減。森夫也只是應了聲嗯，然後繼續日復一日地加班。

當會議終於結束的那一刻，窗外毫無預警地下起了暴雨。

台北的雨總是這樣，來得粗暴又突兀，街燈在濃重的水氣裡被暈染成一片模糊的橘黃。森夫闔上筆電，轉頭看去，娜娜已經在沙發上睡著了。她手裡的手機還亮著，畫面上播著不知名的戀愛綜藝節目。森夫走過去替她關掉螢幕，拉起薄毯蓋在她身上，隨手關掉了客廳的主燈。

婚後的這段日子，他常常有一種奇特而無力的抽離感：這個家明明很溫暖，自己卻還是好累——那是一種連骨頭縫裡都滲著酸痛的疲憊。

按理說，他現在該躺下睡了。

但空蕩蕩的肚子此時卻發出抗議。翻了翻冰箱，裡面只剩半盒板豆腐和一罐過期的優格。他嘆了口氣，抓起鑰匙和雨傘出門，打算去巷口那間徹夜亮著的便利商店買個飯糰填飽肚子。等電梯時，老舊的鏡面鋼板映照出他的模樣：洗得發鬆的 POLO 衫、踩著藍白拖鞋、肩膀微微落枕般地塌陷——一張在台北街頭隨處可見、毫無辨識度的中年男子面孔。

推開大樓鐵門，雨勢稍微小了一些。

他踩在忠孝東路某段連觀光客都不會駐足的斑馬線上。紅燈亮起，他停下腳步。馬路對面是一棟洗石子外牆的老舊公寓，一樓是傳統乾洗店，緊鄰著一扇通往地下室的鐵門，上頭漆著早已剝落大半的褪色綠漆。鐵門上掛著一面手寫的斑駁木板：

**莊武鴛 桌球館**

平常路過，森夫連看都不會多看一眼。

然而這時，一陣冷風將雨絲斜斜地吹過他的脖頸，與此同時，那扇緊閉的鐵門底下——更準確地說，是從潮濕、陰暗的地底深處——毫無預兆地傳來了兩聲極其清脆、乾脆利落的撞擊聲。

乒。

乓。

森夫的腳步瞬間釘住了。

那不是出於好奇。更像是他的身體比大腦更早一步做出了制約反應——他的肩膀猛地沉了下去，右腳本能地往斜前方跨出半步，膝蓋微蹲，重心隨之下壓。那是很多年前，每當體育館裡傳來學長高喊「森夫，接球！」時，全身上下的肌肉自己死死記住的迎擊姿勢。

他就這樣僵在雨中，雨水順著傘骨一顆顆滑落，滴答、滴答，與地下室那規律、激烈的擊球聲奇妙地重疊在一起。

大學一年級，他鬼使神差地報名了桌球社。

當時的教練說他骨架大、下盤穩，是塊打桌球的料，便讓他跟著校隊練了一年。但實際上，那一年裡他多半在幹雜活：撿球、擦球檯、站在場邊看學長們神仙打架。可也就是在那一年，他學會了什麼叫正手握拍，知道了旋轉不是玄學。他至今都記得，在某個昏黃的下午，他與學長連續正手對攻了十個來回沒有漏球，學長拍著他的肩膀說：「欸，森夫，可以喔！」

就因為那輕飄飄的一句話，他整整高興了三天三夜，作夢都在揮拍。

後來呢？後來是忙不完的期末專題、實習、求職。那把貼著姓名貼的球拍被他塞進了書桌最底層的抽屜，任由上面的橡膠皮在不見天日的歲月裡慢慢變黏、變硬、質變。他從不覺得可惜，那時候的他深信，人生有太多比打球更重要、更有價值的事要去追逐。

——那麼現在呢？現在的自己，算得到了那些重要的事嗎？

森夫在暴雨的街頭站了不知道多久，直到乾洗店的老闆娘探出頭來收衣架，用一種看怪人的狐疑眼神打量他時，他才猛然回過神來。

他低著頭嘟囔了一句沒關係，撐好傘，重新邁開步子往便利商店走去。他拿了平日最常吃的芋頭口味三明治，又多抓了一瓶無糖茶。可在櫃檯結帳的時候，那兩個清脆的音節卻像是在他腦海裡安了擴音器一樣，揮之不去。乒、乓。如同有人用指節在胸口不輕不重地叩擊，一下、又一下，敲得他原本空洞麻木的胃袋裡，有什麼沉睡已久的東西正在悄悄甦醒。

回到家時，娜娜依舊睡得很沉。森夫坐在床沿默默地吃著三明治，塑膠包裝的沙沙聲在極度安靜的臥室裡顯得格外清晰。吃完後，他沒有立刻躺下，而是鬼使神差地走向陽台旁，打開了那口落滿灰塵、堆滿雜物的舊儲物櫃。

在最底層、最陰暗的角落。

躺著一只拉鍊已經生鏽發澀的舊球拍袋。他費力地拉開，一股陳舊的橡膠味混雜著樟腦與塵埃的氣味撲鼻而來——極其陌生，卻又叫人鼻酸地熟悉。木質拍柄上，還歪歪斜斜地貼著他大一那年親手寫上的姓名貼，邊角早已泛黃捲曲：**陳森夫**。

膠皮已經徹底乾裂，底下的海綿大概也早已失去了彈性。

森夫緩緩握住那熟悉的拍柄，在黑暗的臥室裡，對著空氣輕輕地空揮了一下。風擦過乾枯的拍面，發出一聲極其細微的切水聲。什麼也沒有擊中，只有虛無的空氣。此時，床上的娜娜在夢裡翻了個身，嘴裡含糊不清地呢喃了一聲：「老公……」

森夫收回手，把廢鐵般的舊球拍輕輕放在自己的膝蓋上，低垂著眼簾，聲音沙啞地輕聲說：

「……明天再說吧。」

窗外的雨，此時下得更大了。

此時的陳森夫並不知道，就在隔壁一堵牆之隔的鄰居許家偉家中，客廳的燈同樣徹夜未眠。許家偉的雙眼正死死盯著螢幕上瘋狂跳動、紅綠交織的期貨 K 線圖，臉色慘白。

這兩個同樣深陷在中年泥沼裡的男人，一個對著乾枯的舊球拍出神，一個在起伏的金融圖表前發呆。此時此刻，他們的命運軌跡還沒有任何交集。

但巷口那間深埋在地底下的 **莊武鴛 桌球館**，那盞昏暗的日光燈，將會一直亮到很晚、很晚。

而某些人看似死水般的人生，往往就是從聽見一聲清脆的「乒」開始，再度命運般地滾動起來的。
