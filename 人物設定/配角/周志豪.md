# 周志豪

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 四十多歲 |
| 身分 | 北岸桌球中心常客；森夫聯盟首戰第一個外館對手 |
| 積分帶 | 約 **1180**（2027/7） |

## 一句話

話不多、反手像牆的老派業餘選手，讓森夫第一次知道外館的「普通對手」也沒那麼普通。

## 與森夫的比賽

- **初登場**：`第十一章｜北岸`
- **結果**：森夫 **3:0 勝**
- **作用**：不是強攻型，而是用穩定與低失誤逼森夫自己把球打對。

## 打法特徵

- 反手貼台穩，幾乎不白送。
- 發球乾淨，不靠花樣取分。
- 正手被調開大角後，還原稍慢半拍。
