# 劉建宏

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 五十出頭 |
| 身分 | 北岸聯盟對手；森夫復出後第一個驗收對手 |
| 積分帶 | 約 **1255**（2027/11） |

## 一句話

護腕、頓半拍發球、球路不花的熟練老手，剛好拿來驗森夫復出後的腳和節奏有沒有真的回來。

## 與森夫的比賽

- **初登場**：`第十五章｜用腳`
- **結果**：森夫 **3:0 勝**
- **作用**：讓森夫確認自己已經能靠腳先到位，而不是靠腰搶位。

## 打法特徵

- 發球前會刻意頓一下，節奏偏慢。
- 球不怪，但很少送分。
- 適合當復出後的第一道檢驗線。
