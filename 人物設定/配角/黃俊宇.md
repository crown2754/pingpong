# 黃俊宇

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 32 歲 |
| 身分 | 業餘桌球愛好者；森夫**第一次正式比賽**的對手 |
| 戰績設定 | 從小打球，無校隊／系隊經歷；館內積分約一千出頭 |

## 一句話

發球花式很多、看起來像高手，基本功卻不紮實——被森夫用賽練看出破綻，卻也在比賽中靠經驗把比分追回來。

## 打法與弱點

### 強項

- **發球花樣多**：下蹲式、勾手、假動作、轉不轉的變化都學過，業餘場上很能唬人。
- **球齡長**：小區球桌、親戚家、朋友館裡泡大，比賽氣氛不陌生。
- **調整快**：前兩局落後後會收花俏、加長球、搶節奏。

### 弱點（森夫逮住的點）

- **基本功不紮實**：腳步亂、還原慢、被調動到底線大角容易漏。
- **發球後第三板**常站位過淺，被森夫短球牽制後來不及退。
- 相持一長，體力與重心問題放大。

## 與森夫第一戰（2026/8/9）

- **賽制**：五局三勝，11 分制。
- **局分**：森夫 2–0 領先 → 黃俊宇連追兩局 → 決勝局 **10–10** 時森夫發「底線延遲」長球出界 → **12–10 黃俊宇勝**。
- **賽後積分**：森夫約 **950**（參賽即得基礎分＋表現加成）；黃俊宇維持千分上下。
