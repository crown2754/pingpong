# 蔡文德

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 五十歲上下 |
| 身分 | 聯盟削球手；森夫在板橋體育館碰上的關鍵對手 |
| 積分帶 | 約 **1378**（2027/12） |
| 持拍 | 左手橫拍；反面偶有顆粒變化 |

## 一句話

慢、沉、耐煩，像一堵會後退也會前壓的舊牆；不是把人打死，而是等人自己先亂。

## 與森夫的比賽

- **初登場**：`第十六章｜削球`
- **結果**：森夫 **3:2 苦勝**
- **作用**：讓森夫第一次正面學會，反擊借不到力時，得自己做旋轉、做節奏、做耐心。

## 打法特徵

- 削球沉，偶爾夾不轉球，專門破壞進攻者判斷。
- 站位離桌遠，但會突然在近台碰短，把節奏切碎。
- 不追求漂亮得分，只逼對手多打一板、多想一拍。

## 劇情功能

- 第十六章的核心關卡。
- 讓森夫把「快」拆成三件事：逼近、換氣、下手。
- 很適合未來回收成森夫再碰一次的技術型老對手。
