# 江柏翰

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 三十出頭 |
| 身分 | 新北聯盟選手；森夫北岸復出戰第二個對手 |
| 積分帶 | 約 **1335**（2027/11） |

## 一句話

反手像牆、正手搶攻慢半拍卻夠硬的中階選手，讓森夫知道「腳回來了」不等於「手也回來了」。

## 與森夫的比賽

- **初登場**：`第十五章｜用腳`
- **結果**：森夫 **1:3 負**
- **作用**：讓森夫在復出後看清自己輸的已不是腰，而是球質與還原。

## 打法特徵

- 反手貼台穩，正手節奏不快但能接續壓制。
- 會用長球頂住反手，逼對手退到底線。
- 屬於「不華麗但很完整」的聯盟中段對手。
