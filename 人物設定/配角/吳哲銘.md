# 吳哲銘

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 三十多歲 |
| 身分 | 板橋體育館聯盟對手；森夫第十六章第一個對手 |
| 積分帶 | 約 **1295**（2027/12） |

## 一句話

看起來普通、打起來不鬆的標準外館選手，專門懲罰那些想在嘈雜場館裡用力證明自己的人。

## 與森夫的比賽

- **初登場**：`第十六章｜削球`
- **結果**：森夫 **3:1 勝**
- **作用**：把板橋的觀眾壓力先壓到森夫身上，逼他先把自己安靜下來。

## 打法特徵

- 發球乾淨，反手穩定，站位正。
- 不主動花俏搶分，擅長等對手先亂。
- 屬於「看起來不兇，實際很煩」的對手型。
