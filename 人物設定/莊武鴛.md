# 莊武鴛

| 範本 | 莊智雄 |
|------|--------|
| 身分 | **莊志淵**的哥哥；**莊武鴛桌球館**創辦人／櫃檯與教練 |
| 年齡 | 五十多歲（比弟弟大約十歲） |
| 打法（年輕時） | 橫拍，實戰派；國內賽事經驗，未走弟弟的國際賽路線 |

## 一句話

花白頭髮、記帳、裁邊器、一句話點醒人的館主；森夫以為只是姓氏巧合，其實是莊志淵的哥哥。

## 在故事中的功能

| 功能 | 說明 |
|------|------|
| 館的靈魂 | 地下室球館不是背景，是有人撐起來的家 |
| 賽練推手 | 主張「對牆不如賽練」「比賽從發球開始」 |
| 長輩參考 | 原型莊智雄：打過球、懂現實、不灌雞湯 |
| 弟弟的影子 | 牆上莊志淵照片、獎盃；偶爾一句「我弟那種球你學不來，發球可以學」 |

## 與森夫

- 第一次見面（5/16）就記住他「大學打過」；基礎班（6/18）正式介紹賽練與交流賽。
- 不會過度鼓勵，常用務實話：**第一場比賽不是為了贏，是為了知道輸在哪裡。**
- 看森夫鑽發球、寫筆記，會說：「你像工程師。很好，桌球也需要 debug。」
- **不錄影**。館裡忙，頂多遠遠看兩眼、記帳、順口點一句；轉介復健時靠口述，不會整理學員影片。

## 說話風格

- 短句、平實、偶爾冷幽默。
- 「裁邊免費教你。」「三十五？我弟三十五還在歐洲挨罵呢，你怕什麼。」
- 學員喊他 **莊哥**。
