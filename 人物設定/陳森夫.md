## 一句話

三十五歲的社畜工程師，在大學桌球社邊緣待過一年，十幾年後被路過球館的擊球聲喚醒，決定在工作之外重新找回「自己還活著」的證明。

## 大學時期（回憶殺）

- 大一報名桌球社，被校隊教練看中體質，跟著校隊訓練約一年。
- 實際上是邊緣人：幫學長撿球、基本對攻會漏，談不上主力。
- 那一年留下的底子：正規握拍、知道旋轉是什麼、對過幾次正手對攻與簡單步伐。
- 大學專題(大三)與(大四實習)就業壓力來襲，球拍收進抽屜，漸漸不再碰球。

## 現在的日常

- 遠端與進辦公室混著做，需求變更、上線前夕、跨時區會議是常態。
- 對著螢幕寫 code 的時間比對人說話多；回家常只剩滑手機的力氣。
- 外表普通、不搶眼，同事眼裡是「好配合、不太抱怨」的那種人。
- 內心對重複與虛無有感，但習慣壓下去，用加班與已讀不回撐過每一天。

## 性格

- **表面**：內斂、話不多、遇事先忍。
- **內在**：一旦認定某件事，執著程度嚇人；不服輸，但不太會講漂亮話。
- **實驗控**：對器材有近乎偏執的興趣，會不斷更換膠皮、底板，研究硬度、海綿、顆粒；寫 code 的 A/B test 心態原封不動搬到球拍上。
- **優勢**：專注、能吃苦、肯研究（寫 code 練出來的）；三十五歲的耐心與算計，彌補體能劣勢。
- **弱點**：起步體重偏重、體力差、關節僵硬、容易在怪打法面前吃旋轉；被生活磨掉的自信，要重新一點一點撿回來。
- **身體變化**：練球三年間從 95 kg 瘦到 80 kg，是劇情裡看得見的「社畜重啟人生」證明之一。

## 與桌球

- **起點**：路過（或聽見）球館擊球聲 → 翻出泛黃舊球拍 → 先從社區球桌與許家偉練起。
- **目標弧線**：社區練球 → 球館基礎訓練 → 業餘積分賽 → 三年後衝上 **1800 分** → 與國手對決。
- **打法傾向**（可隨劇情調整）：正手為主、重落點與算球，不靠年輕人的連續爆衝硬拚。
- **器材實驗**：球技還在爬，裝備先換了一輪——狂飆、蝴蝶、各種硬度膠皮、甚至研究灌油；常常練兩週就換組合，教練或球友吐槽「人沒進步，拍子進步了」。家偉有時受不了他的器材焦慮，有時也蹭來試打。

## 與許家偉的關係

- 同棟鄰居，國小到高中都隔壁班，本來只是點頭之交。
- 森夫拉他上社區球桌後，家偉漸漸上頭；森夫是發起者，家偉是「被帶進坑」卻越陷越深的那個。
- 家偉是徹底桌球小白，森夫勸他改打橫拍，他堅持要中直——森夫常為此傻眼。
- 家偉會主動問「什麼時候要打？」；球剛能打進台內時，森夫忍不住誇他「越來越好、球質不錯」。
- **誇獎陷阱**：家偉一被誇就以為要「再加力證明自己」，越打越大力，球全飛出台外；森夫當場傻眼，誇也不是、不誇也不是。
- 家偉後期同樣達到 **1800 分**；兩人可寫成「鄰居內戰」「積分賽碰頭」的對手戲。

## 說話與習慣

- 用語偏直白，不熱血喊口號；緊張時會講半句就停。
- 習慣把情緒藏在動作裡（多練一組、默默換膠皮），而不是講出口。
- 對程式與桌球都會鑽細節；換膠皮時會認真做筆記，像記 bug log 一樣記「這片太彈、那片吃球」。
- 對人比較笨拙，但跟家偉講器材可以講很久，對方嗯一聲他還能再講十分鐘。

## 劇情備註

- 雙結局共通：無論輸贏，最後都進入球館工作（教練／行政），找到下半生的歸宿。
- 結局 A（輸）：雖敗猶榮，體力不支但贏得尊重。
- 結局 B（贏）：奇蹟一勝後氣力放盡，無法再高強度對抗，但無悔。
