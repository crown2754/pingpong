# 許家偉

## 基本資料

| 項目 | 內容 |
|------|------|
| 年齡 | 35 歲（與陳森夫同年，稍大幾週） |
| 生日／星座 | 3 月 31 日；**牡羊座** |
| 身高／體重 | 175 cm；約 75 kg，練球期間大致維持 |
| 職業 | 在家炒股玩期貨 |
| 感情 | **單身一輩子**（無配偶、無交往對象） |
| 居住地 | 台北市，與陳森夫同棟 |
| 關係 | 鄰居、死黨；國小到高中都隔壁班；最初被拉來練球的陪打 |
| 握拍 | **中直**（森夫勸過改橫拍，聽完還是堅持中直） |

## 一句話

平時安靜到像不存在，被森夫拖進桌球坑後越來越上頭；從連台都打不進，到與森夫同登 **1800 分**——中直到底，悶聲幹大事。

## 性格核心

### 悶騷

- 外表話少、表情淡，不熟的人以為他沒在聽。
- 心裡戲很多，偶爾冷不防一句吐槽精準到讓森夫接不住。
- 不主動社交，但對認定的人會用「行動」表態（準時出現球桌、默默買新球、錄影回放）。

### 越來越上頭

- 一開始只是被森夫拜託「陪我打一下」，姿勢亂、當發球機。
- 漸漸自己查影片、買裝備、記掛積分賽日期；森夫還在猶豫要不要報名，他已經把行事曆空出來了。
- 上頭表現：練完不說累、輸球回家還在空揮、開始研究對手打法——但嘴上仍裝沒事。

### 不能誇（會誤讀成「要更用力」）

- 嘴上被稱讚仍會否認、裝沒事；但**技術上的誇獎**他聽進去了，只是理解錯方向。
- 球剛能打進台內那陣子，森夫誇他「打得越來越好、球質越來越不錯」——家偉的下一句往往是沉默，然後下一球**大力出奇蹟**，直接打飛出台外。
- 他不是故意搞事，是真的以為：球質好 = 要再加力；結果從「打不進台」變成「打出台」。
- 森夫每次都傻眼，後來學乖想改誇「落點穩」「控制住了」，家偉有時還是會突然加力——這條笑料線可以貫穿前期章節。
- **淘汰賽掉鍊子**：積分能衝上 1800，但一到淘汰賽關鍵局，觀眾或對手一誇、教練一肯定，他就忍不住加力，球打飛、心態崩，**常常早退**——永遠差那麼一步上大場面。

## 在故事中的功能

| 功能 | 說明 |
|------|------|
| 吐槽擔當 | 用冷靜視角戳破森夫的自我感動或社畜自嘲 |
| 陪練起點 → 同級對手 | 社區球桌階段是小白陪打；後期兩人皆達 **1800 分**，可同場積分賽、互為勁敵 |
| 情緒錨點 | 森夫想放棄時，家偉不會講大道理，只會傳「今晚七點，老地方」 |
| 讀者代入 | 一般人不會像森夫那樣有大學社團底子；家偉代表「純被拉進坑」的路線 |

## 與陳森夫的互動模式

- **森夫**：熱起來時話變多，容易鑽技術細節。
- **家偉**：點頭、嗯、偶爾一句補刀；真正在意的事會記很久（例如森夫某次說「算了不打了」）。
- 兩人不需要煽情告白友誼；默契在「球桌邊多站十分鐘」和「幫對方撿球不講話」。
- **感情對照**：家偉**單身一輩子**，桌球與交易填滿生活；森夫有娜娜，回家有人等——兩條人生線並行，不互相可憐，只是不一樣。

## 桌球相關

- **起點**：**完全小白**，比零基礎還零基礎；森夫怎麼教握拍，他都要打中直。
- **器材（第一季）**：長期借用森夫大學舊中直；後期**偷偷自購日直**，起初生硬，後來很快學會**握筆握拍（日直）**；森夫本想笑他，結果被他兩板打到閉嘴。
- **初期慘況**：連球都打不進台內，森夫撿球撿到懷疑人生。
- **主動性**：嘴上悶，一上頭就會主動問森夫「什麼時候要打？」「今晚有空嗎？」——比森夫還積極。
- **成長曲線（前期）**：
  1. 打不進台內 → 偶爾擦邊進台 → 能穩定過網落台
  2. 森夫一誇 → 大力出飛球 → 森夫傻眼 → 反覆循環
  3. 漸漸學會「力發在球上而不是打死球」（過程可寫得慢、好笑）
- **成長曲線（中後期）**：
  - 固執**中直**到底，不靠森夫那套橫拍器材實驗，用同一套拍慢慢磨出穩定球質。
  - 與森夫同期打積分賽，進步曲線更陡——從完全小白到 **1800 分**，是故事裡最不可能的弧線之一。
  - 1800 後可與森夫同場積分賽、互有勝負；中直怪招在業餘場上常讓對手吃驚。
  - **國手之戰與他無緣**：只有森夫下場；家偉多半已在淘汰賽自己作死出局，可在場邊看，或傳訊息吐槽。

## 說話風格範例

- 「……哦，你要打就去打，我沒要幹嘛。」（其實已經換好球鞋）
- 「橫拍怪怪的。中直比較順。」（森夫放棄勸說的表情可以寫一整段）
- 「今晚幾點？我先把單子掛完。」
- 「沒有很好啦。」（下一秒全力一板，球飛出三米外）
- 「再一局。最後一局。」（然後永遠不是最後一局）
- 「你昨天說不打了。我球拍都帶了。」
- （淘汰賽後）「……他們說我打得很好。」（下一場全力一板出界，賽季結束）
