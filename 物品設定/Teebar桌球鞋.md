# Teebar 桌球鞋

## 基本資料

| 項目 | 內容 |
|------|------|
| 品牌 | [Teebar](../品牌設定/Teebar.md)（範本 Tibhar） |
| 類型 | 桌球專用運動鞋 |
| 特色 | **免綁鞋帶**（一腳蹬鞋舌／彈性鞋口，或旋鈕固定系統） |
| 森夫入手 | 第二季（外館比賽前後） |

## 在故事中的功能

- **實用**：膠地啟動、小碎步時不易打滑；比賽間不用彎腰綁鞋帶，減少分心。
- **象徵**：第二季從「館內」走向「外戰」，裝備從只換拍延伸到**腳下**。
- **夫妻戲**：娜娜嫌貴，但認可免綁帶對森夫老腰友善；可與及一卡哇談判並列。

## 與劇情連結

| 階段 | 事件 |
|------|------|
| 第二季初 | 北岸外館打滑／鞋帶鬆脫，萌生換鞋念頭 |
| 第二季中 | 與紅張同期或稍後入手；腳步穩定度提升 |
| 第十二章 | 娜娜談判：QKA 玩偶換桌球鞋一半預算；紅張下週取貨 |
| 後續 | 長線體能線（積分 1600→1800）可持續穿著 |

## 正文用語

- 品牌寫 **Teebar**；可描述「免綁鞋帶」「一蹬進去」「旋鈕鎖緊」等，不強調現實型號。
- 價格區間：中高；比入門運動鞋貴，比頂級底板便宜——仍會觸發娜娜預算警報。
