# LIME

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 即時通訊 App（小說世界觀中的 LINE 對應） |
| 外觀 | 綠色主題、對話泡泡、已讀標記、貼圖商店 |
| 用途 | 角色日常聯絡、約球、傳連結與截圖 |

## 在故事中的用法

- 正文中統稱 **LIME**，不直接使用現實品牌名。
- 森夫與許家偉的對話以簡短、已讀、省略見長——符合兩人悶騷個性。
- 可呈現：傳 YouTube 連結、球拍截圖、約球時間、「嗯」「哦」類回覆。
- 娜娜與森夫亦用 LIME，但頻率較低，多為生活瑣事或撒嬌貼圖。

## 格式建議（正文）

```
【LIME｜許家偉】

```

## 備註
