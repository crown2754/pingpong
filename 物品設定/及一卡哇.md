# 及一卡哇

## 基本資料

| 項目 | 內容 |
|------|------|
| 類型 | 日本萌系 IP（小說世界觀中的 Chiikawa 對應） |
| 角色 | 小可愛們、吉伊、小八、烏薩奇等（正文可用「及一卡哇角色」統稱） |
| 周邊 | 玩偶、吊飾、文具、展覽限定品 |

## 在故事中的功能

- **林娜娜**喜愛的 IP；森夫用「陪去展覽、買及一卡哇」換取器材預算，是夫妻間的甜蜜談判籌碼。
- **華山**常辦快閃或特展；森夫答應帶娜娜去，作為請款條件之一。

## 與劇情連結

| 章節 | 事件 |
|------|------|
| 第二章 | 森夫請款買球拍，答應華山及一卡哇展 + 買周邊 |
| 待定 | 國手戰前娜娜拿及一卡哇吊飾當幸運物等 |

## 正文用語

- 統稱 **及一卡哇**，不直接使用現實 IP 名稱。
- 可寫：吉一、小八類角色暱稱，但以「及一卡哇」為主稱。

## 備註

- 價格區間：展覽周邊從幾百元到限量上千元不等，適合寫「森夫心疼但甘願」的對照。
